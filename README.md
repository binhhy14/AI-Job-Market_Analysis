# AI Job Market Analysis

This project performs **Exploratory Data Analysis (EDA)** on an AI and Data Science job market dataset.  
The goal is to explore hiring trends, salary distributions, required skills, and job demand across industries and countries.

---

# Dataset

The dataset contains job postings related to AI and Data Science roles, including:

- Job title
- Salary
- Country
- Industry
- Work type (Remote / Hybrid / Onsite)
- Years of experience
- Technical skills (Python, SQL, Machine Learning, Deep Learning, Cloud)

---

# Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# Analysis Performed

The analysis explores several aspects of the AI job market:

- Job role distribution
- Salary distribution
- Experience distribution
- Skill demand analysis
- Salary by job role
- Salary by work type
- Hiring industries
- Job distribution by country

---

# Key Insights

Some important findings from the analysis:

- **AI Engineers and Machine Learning Engineers show the highest salary ranges.**
- **Python and SQL are among the most demanded technical skills.**
- The **Technology industry has the highest number of job postings.**
- Salary ranges are **similar across Remote, Hybrid, and Onsite work types.**
- Demand for AI and data roles appears **globally across multiple countries.**
